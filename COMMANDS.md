### 📂 (Navigation & File System)

* **"cd"** - change directory, jumping between folders  
    * `"cd file_name"` - entering exact folder

* **"dir"** - shows whats inside (similar to ls)

* **"dir /a"** - shows whats inside + hidden files (ls -a)

* **"dir /s"** - shows whats inside current folder and every subfolder inside it

### 📄 (Content Extraction)

* **"type"** - to open file (similar to cat)

### ⚙️ (Other commands)

* **"cls"** - clear screen

### 🖥️ (System & Network)

* **"whoami"** - shows current user

* **"hostname"** - shows the name of computer
  
* **"systeminfo"** - displays information about system (OS, processor, RAM etc)

*  **"ipconfig"** - shows network cofiguration (ip address, gateway etc.)
